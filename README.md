## Installation

You can install the package via composer:

```bash
composer require phpcodertop/kg-to-pound
```

## Usage

``` php
$skeleton = new Phpcodertop\KgToPound();
echo $skeleton->echoPhrase('Hello, Phpcodertop!');
```
